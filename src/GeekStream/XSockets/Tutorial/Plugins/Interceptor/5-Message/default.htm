PreConditions

  1. You have installed XSockets. Setup Example or anything equal to this
  2. If you are NOT using the XSocketHandler project that is installed with XSockets
    • See to it that the assembly get copied to the pluginfolder (see XSocketHandler postbuild events)
  3. Add a refernce to XSockets.Extensibility.Interceptors.dll (found under XSocketServerPlugins)

Code - MessageInterceptor

  1. We just create a basic MessageInterceptor by using the Scaffolders (available for all interceptor types)
    1. Open up the Package Manager Console
    2. Write: Scaffold XMessageInterceptor MyMessageInterceptor
    3. This will create a new MessageInterceptor name MyMessageInterceptor
    [Export(typeof(IMessageInterceptor))]
public sealed class MyMessageInterceptor : XBaseInterceptor, IMessageInterceptor
{
    public void Incomming(IXBaseSocket socket, string data)
    {
        //Implement your interceptor stuff here
    }

    public void Outgoing(IXBaseSocket socket, string data)
    {
        //Implement your interceptor stuff here
    }
}
