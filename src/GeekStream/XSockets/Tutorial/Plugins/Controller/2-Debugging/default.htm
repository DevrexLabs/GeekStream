Since the default development server does not support debugging your plugins you will have to setup your own developmentserver. The steps to do this is listed below.
NOTE: We will create a nuget package for a debugserver "XSockets.DebugServer", but if you ever need to to it manually this is how.

PreConditions

  1. You have installed XSockets. Setup Example or anything equal to this

Steps - Setup DebugServer

  1. Create a new ConsoleApplication and name it DebugServer
  2. Change target framework to .NET 4 by selecting properties and the applicaiton tab on your new project.
    Then change the target from .NET Framework 4 Client Profile to .NET Framework 4
  3. Add references to XSockets.Core and XSockets.DevelopmentServer
  4. Copy the config-content from the developemtserver to the config in your new project.
  5. Change to RefPath to point to the folder above your plugindirectory (example below) 
    //My plugins dir (XSocketServerPlugins) is located directly under the path below.
<add key="RefPath" value="C:\Projects\XSockets.NET\Dev\XSockets" />
  6. Now paste this code into the Main of your program (in the ConsoleApplication) 
    try
{                      
    new Start();
}
catch (Exception ex)
{
    System.Console.WriteLine(ex.Message);
    if (ex.InnerException != null)
        System.Console.WriteLine(ex.InnerException.Message);
    System.Console.ReadLine();
}
